Monitor de Estado de IP

Este es un proyecto de monitorización de infraestructuras de red. Incluye:

  • Panel de servidores
  • Historial de fallas
  • Control de acceso con login
  • Reportes en PDF

Para ejecutar el proyecto completo necesitas un entorno Python/Django. Inicia el servidor en monitor_infra y luego usa el botón para abrir la app.
